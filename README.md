## Jord-simulator

Dette er mitt prosjekt i emne TDT4102 ved NTNU

Målet mitt er å simulere jorda, med værsystemer, båter og fly

Min første utfordring er å lage min egen 3d rendrer med AnimationWindow bibilioteket til TDT4102 #CHECK

Min neste utfordring er å gjøre kuben til en kule med ønsket antall trekanter #CHECK

Min neste utfordring blir å tegne jorda på denne kulen.
